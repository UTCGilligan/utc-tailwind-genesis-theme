# Contributing

UTC is a starter child theme for people making websites with the [Genesis Framework](https://my.studiopress.com/themes/genesis/) for WordPress. Thank you for your interest in contributing!

## File issues

Contribute bug reports and feature suggestions as [GitHub Issues](https://github.com/UTCbridget/utc-tailwind-genesis-theme/issues).

## Contribute code

We recommend that you first [create an issue](https://github.com/UTCbridget/utc-tailwind-genesis-theme/new) to propose major changes before working on them.

[Fork](https://help.github.com/en/articles/fork-a-repo) the [UTC repository](https://github.com/studiopress/utc) to your own GitHub account, then clone it to work on locally:

    git clone git@github.com:you-username/utc.git

Create a new branch to work on your changes. New branches should be forked from the `develop` branch.

    git checkout -b feature/your-new-feature

Make your changes and commit your code.

When you're ready, [create a pull request](https://github.com/UTCbridget/utc-tailwind-genesis-theme/pulls) to outline and contribute your changes.
